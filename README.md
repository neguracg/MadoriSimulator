# 間取りシミュレーター (Madori Simulator)

方眼紙感覚で住宅の間取りをざっくり組み替えながら、部屋の配置・広さ・全体面積を試行錯誤するためのブラウザツール。精密なCADではなく「変更の容易さ」を重視した概算ツールです。

## 主な機能

- **455mmグリッド**（910mmごとに太線）上で部屋ブロックを作成・編集
- 部屋は「占有マスの集合」として管理し、面積（㎡ / 畳 / 坪）を自動計算
- **編集モード / 移動モード** の2モード
  - マス追加・マス削除、角・辺ドラッグでの伸縮
  - 移動は重なりを許容し、編集モードに戻った時点で確定（上のレイヤー優先・分断時は自動分割）
- 種別ごとの自動色分け（種別の追加・色変更可）
- 外壁ラインの自動追従、各辺の長さ表示（m）
- 1階・2階の切替、階別／延床面積サマリー
- 複数の間取りをタブで保持（オートセーブ）
- Undo / Redo、JSONエクスポート・インポート

## 開発

```bash
npm install
npm run dev      # 開発サーバ
npm run build    # 本番ビルド (dist/)
```

React + TypeScript + Vite。
